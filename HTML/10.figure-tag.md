# figure 태그
`figure` 태그는 이미지나 테이블 같은 컨텐츠를 표현할 때에 사용한다. `figure` 태그는 자식 태그로 `figcaption` 태그를 가질 수 있다. `figcaption` 태그는 컨텐츠의 이름을 의미하는 태그이다.

`figcaption` 태그는 컨텐츠의 위나 아래에 위치할 수 있다.

```html
<figure>
  <figcaption>테이블 이름</figcaption>

  <table>
    <tr>
      <td>1</td>
      <td>2</td>
    </tr>
    <tr>
      <td>3</td>
      <td>4</td>
    </tr>
  </table>
</figure>
```
