# form 요소 그룹화하기
서로 연관이 있는 `form` 요소들을 묶을 수 있다. `fieldset` 태그와 `legend` 태그를 사용한다.

```html
<fieldset>
  <legend>필드셋</legend>

  <div>
    <label>
      입력 필드 1: <input type="text">
    </label>
  </div>
  <div>
    <label>
      입력 필드 2: <input type="text">
    </label>
  </div>
  <div>
    <label>
      입력 필드 3: <input type="text">
    </label>
  </div>
</fieldset>
```
