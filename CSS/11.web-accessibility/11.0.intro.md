# 웹 접근성이란?
웹을 사용하는 다양한 사람들 중에는 어딘가 불편한 사람들도 있다. 눈이 불편한 사람들이거나 듣지 못하는 사람들이 있다. 그런 사람들을 위해 사이트들은 사용자 경험에 있어서 차별 없이 같은 컨텐츠를 받아들일 수 있게 만들어야만 한다.

웹 접근성은 웹 표준과는 다른 것으로, 웹 접근성이 보장된 사이트는 많은 것을 고려했다는 생각을 하게 만들 수 있다.
