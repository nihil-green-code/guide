# 텍스트 스타일
이 문서에서는 텍스트의 스타일을 바꿀 수 있는 `font-style` 속성에 대해 알아본다. 기울기를 줄 때에 많이 사용한다. 간단하게 알아보고 넘어가도록 하자.

```css
p.italic {
  font-style: italic;
}
```

```html
<p>기울기 적용 안됨.</p>
<p class="italic">기울기 적용됨.</p>
```
